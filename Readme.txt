

Set up:
*********************************

Required: nodejs and npm

open command prompt in project folder

install mime-types:
in command prompt type: >npm install mime-types


run the server: >node server.js


Run website:
***********************************

Server will run on port 2406:

Open browser: access local host via port 2406:
  + http://127.0.0.1:2406/
  + http://localhost:2406/

Choose the hero to view via the drop down menu and click View button to
get the information.

(Short-cut: you can press space to change hero and press enter to view)
